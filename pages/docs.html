Quick Start

1️⃣

Create an Account

Register on iq4u8 and wait for admin approval. Once approved, you can log in to your dashboard.

2️⃣

Open the browser-based terminal to run commands directly on your VPS. No SSH client needed.

3️⃣

Upload Files

Use the built-in file manager to upload your project files, or use the terminal to clone from Git.

4️⃣

Deploy Your App

Run your application using the terminal. Support for Node.js, Python, and more.
